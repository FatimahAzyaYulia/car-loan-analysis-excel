# 🚗 Car Loan Dashboard (Microsoft Excel)

## 📌 Project Overview

This project presents an interactive Excel dashboard for analyzing car loan application data from **2020 to 2023**. The dashboard provides key metrics and visualizations to help understand loan trends, approval status, regional performance, and loan type distribution.

The dashboard was built using Microsoft Excel with PivotTables, PivotCharts, Slicers, and business insights to support data-driven decision making.

---

## 🎯 Project Objectives

This project aims to answer the following business questions:

- ## Business Questions

This dashboard is designed to answer the following business questions:

1. What is the total number of loan applications?
2. What is the total loan amount issued?
3. How do loan applications trend by year?
4. How do loan applications trend by month?
5. What is the distribution of loan approval statuses?
6. Which regions have the highest number of loan applications?
7. Which loan types are most frequently selected by customers?

## 📈 Key Insights

📈 Trend: Loan applications peaked in 2021 with 2,551 applications before gradually declining.
🌍 Regional Performance: North recorded the highest number of loan applications (2,046).
🚗 Customer Preference: Used Car Loan is the most popular financing product with 3,376 applications.
📋 Approval Status: Pending has the highest number of applications (3,372), indicating an opportunity to improve processing efficiency.
💡 Recommendation: Strengthen marketing efforts for Used Car Loan while reducing the backlog of Pending applications.
